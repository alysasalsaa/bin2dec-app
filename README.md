bin2dec-app

Binary to Decimal converter — App Ideas Collection Tier 1 Beginner project.

🔢 About

Enter up to 8 binary digits (0s and 1s) and instantly see the decimal equivalent. Built as part of a personal challenge to complete all projects in the App Ideas Collection.

✅ User Stories


 User can enter up to 8 binary digits in one input field
 User is notified if anything other than a 0 or 1 was entered
 User views the decimal (base 10) equivalent of the binary number entered
 Bonus: User can enter a variable number of binary digits


🛠 Tech Stack


HTML
CSS
JavaScript (vanilla)


⚙️ Constraints followed


No arrays used to store binary digits
Decimal conversion uses a single mathematical function (Math.exp with Math.LN2)


🚀 How to run

Just open index.html in your browser — no build tools needed.
