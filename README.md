# bin2dec-app

A simple Binary to Decimal converter built with HTML, CSS, and JavaScript.

## About

Enter up to 8 binary digits (0s and 1s) and instantly see the decimal equivalent.

## Features

- [x] Enter up to 8 binary digits
- [x] Shows error if input contains anything other than 0 or 1
- [x] Displays the decimal equivalent in real-time
- [x] Visual bit display for each digit entered
- [x] Supports variable number of binary digits (bonus)

## Tech Stack

- HTML
- CSS
- JavaScript (vanilla)

## How to run

Open `index.html` in your browser — no build tools needed.
